# Form Types <Badge type="feature" text="3.13+" /> <Badge type="addon" text="Add-on" />

Freeform 3.13 introduces the concept of form types. Form types are additional add-on plugins that add more specialized behavior and features to your forms. Form type add-ons can be installed on any edition of Freeform.
